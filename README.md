# Boutique Method — GitHub Pages (pronto)

## Como publicar
1) Crie um repositório no GitHub (ex: boutique-site)
2) Faça upload de TODOS os arquivos deste ZIP na raiz do repo
3) Settings → Pages → Deploy from branch → main / (root)

## Formulário
GitHub Pages não tem backend.
- Crie um endpoint no Formspree e troque o action em diagnostico.html
- Troque o _redirect para a URL real do seu GitHub Pages
